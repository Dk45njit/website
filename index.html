This assignment is about learning to host websites using Github Pages using linux commands, Vi Text editor and Git

Linus Commands

  • CD - Change Directory
  • MKDIR - Make Directory
  • LS - List Files

Vi Commands

  • I - Insert Mode
  • Esc - Get out of insert mode
  • Shift key + Colon - Command prompt for vi
  • W - Command to write
  • Q - Command to quit
  • WQ - Write and Quit

Git Commands

  • Git init - create repository
  • Git add Filename - Add file to Git
  • Git commit -m "message" - Do the commit to save changes
  • Git checkout -b BranchName - Make a new branch
  • Git merge BranchName - Merge the branch
